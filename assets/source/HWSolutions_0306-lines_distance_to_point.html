  1. Find the distance from the given point \(P\) to the given line \(l\):

    1. \(P=(1,2)\) and \(l\) is \(2x+3y+5=0\)

      The line is in general form, so \(A=2,\) \(B=3,\) \(C=5,\) \(x_1=1,\) and \(y_1=2.\) Thus the distance is \(\frac{|(2)(1)+(3)(2)+(5)|}{\sqrt{2^2+3^2}}=\frac{13}{\sqrt{13}}=\sqrt{13}.\)

    2. \(P=(1,1)\) and \(l\) is \(x-y=0\)

      The line is in general form, so \(A=1,\) \(B=-1,\) \(C=0,\) \(x_1=1,\) and \(y_1=1.\) Thus the distance is \(\frac{|(1)(1)+(-1)(1)+(0)|}{\sqrt{1^2+(-1)^2}}=\frac{0}{\sqrt{2}}=0.\)

      Alternatively, notice that the point \((1,1)\) is on the line \(x-y=0\), and so the distance from the point to the line has to be \(0.\)

    3. \(P=(3,-2)\) and \(l\) is \(4x-2y+8=0\)

      The line is in general form, so \(A=4,\) \(B=-2,\) \(C=8,\) \(x_1=3,\) and \(y_1=-2.\) Thus the distance is \(\frac{|(4)(3)+(-2)(-2)+(8)|}{\sqrt{4^2+(-2)^2}}=\frac{24}{\sqrt{20}}=\frac{12}{\sqrt{5}}.\)

    4. \(P=(3,-2)\) and \(l\) is \(y=-\frac{1}{3}x+5\)

      The line is not in general form, so we set it equal to zero: \(\frac{1}{3}x+y-5=0.\) Now \(A=\frac{1}{3},\) \(B=1,\) \(C=-5,\) \(x_1=3,\) and \(y_1=-2.\) Thus the distance is \(\frac{|(1/3)(3)+(1)(-2)+(-5)|}{\sqrt{(1/3)^2+2^2}}=\frac{6}{\sqrt{37/9}}=\frac{18}{\sqrt{37}}.\)

    5. \(P=(-2,-5)\) and \(l\) is \(y=5\)

      The line is not in general form, so we set it equal to zero: \(0x+1y-5=0.\) Now \(A=0,\) \(B=1,\) \(C=-5,\) \(x_1=-2,\) and \(y_1=-5.\) Thus the distance is \(\frac{|(0)(-2)+(1)(-5)+(-5)|}{\sqrt{0^2+1^2}}=\frac{10}{\sqrt{1}}=10.\)

      Alternatively, this is the horizontal line \(y=5\) and the point \((-2,-5)\) is \(10\) units directly below this line. Thus \(10\) is the distance from the point to the line.

  2. Suppose \(P=(1,1)\) and \(l\) is \(y=-x+5\).

    1. Find an equation of the line \(m\) perpendicular to \(l\) that goes through \(P\).

      The line \(l\) has slope \(-1\), so the line \(m\) perpendicular to \(l\) has slope \(1.\) This line also goes through \((1,1).\) So the line perpendicular to \(y=-x+5\) through \((1,1)\) is \(y-1=1(x-1).\) This simplifies to \(y=x.\)

    2. Find the coordinates of the intersection point \(Q\) of line \(m\) with line \(l\).

      We must find the intersection point of the lines \(y=-x+5\) and \(y=x.\) Combining the equations, we get \(x=-x+5.\) Thus \(2x=5,\) and so \(x=\frac{5}{2}.\) Thus the intersection point is \((\frac{5}{2},\frac{5}{2}).\)

    3. Use the distance formula to find the distance from point \(P\) to point \(Q\).

      The distance is \(\sqrt{(5/2-1)^2+(5/2-1)^2}=\sqrt{18/4}=\frac{3\sqrt{2}}{2}.\)

    4. Use the distance from a point to a line formula to find the distance from point \(P\) to line \(l\).

      Let us put \(y=-x+5\) in general form: \(x+y-5=0.\) Now the distance is \(\frac{|(1)(1)+(1)(1)+(-5)|}{\sqrt{1^2+1^2}}=\frac{3}{\sqrt{2}}=\frac{3\sqrt{2}}{2}.\)
