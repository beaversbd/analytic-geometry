  1. Find the slope, inclination (rounded to the nearest degree), and intercepts of the following lines and then graph.

    1. \(y=2x+1\)

      This equation is in slope-intercept form so the slope is \(m=2\). Thus the inclination is \(\tan^{-1}(2)\approx 63^\circ.\) Since this is in slope-intercept form, we see that the \(y\)-intercept is \((0,1).\) Solving \(0=2x+1\) gives us the \(x\)-intercept \((-\frac{1}{2},0).\)

    2. \(y=3x-3\)

      This equation is in slope-intercept form so the slope is \(m=3\). Thus the inclination is \(\tan^{-1}(3)\approx 72^\circ.\) Since this is in slope-intercept form, we see that the \(y\)-intercept is \((0,-3).\) Solving \(0=3x-3\) gives us the \(x\)-intercept \((1,0).\)

    3. \(y=-\frac{1}{2}x+2\)

      This equation is in slope-intercept form so the slope is \(m=-\frac{1}{2}\). Thus the inclination is \(\tan^{-1}(-\frac{1}{2})+180^\circ\approx 153^\circ.\) Since this is in slope-intercept form, we see that the \(y\)-intercept is \((0,2).\) Solving \(0=-\frac{1}{2}x+2\) gives us the \(x\)-intercept \((4,0).\)

    4. \(y=\frac{3}{4}x-\frac{1}{2}\)

      This equation is in slope-intercept form so the slope is \(m=\frac{3}{4}\). Thus the inclination is \(\tan^{-1}(\frac{3}{4})\approx 37^\circ.\) Since this is in slope-intercept form, we see that the \(y\)-intercept is \((0,-\frac{1}{2}).\) Solving \(0=\frac{3}{4}x-\frac{1}{2}\) gives us the \(x\)-intercept \((\frac{2}{3},0).\)

  2. Find the slope, inclination (rounded to the nearest degree), and intercepts of the following lines and then graph.

    1. \(2x+3y-6=0\)

      This equation is in general form so it is easiest to find the intercepts first. When \(y=0\), we get \(2x-6=0\), and so the \(x\)-intercept is \((3,0).\) When \(x=0\), we get \(3y-6=0\), and so the \(y\)-intercept is \((0,2).\) Thus the slope is \(m=\frac{2-0}{0-3}=-\frac{2}{3}\). Thus the inclination is \(\tan^{-1}(-\frac{2}{3})+180^\circ\approx 146^\circ.\)

    2. \(x-2y-3=0\)

      This equation is in general form so it is easiest to find the intercepts first. When \(y=0\), we get \(x-3=0\), and so the \(x\)-intercept is \((3,0).\) When \(x=0\), we get \(-2y-3=0\), and so the \(y\)-intercept is \((0,-\frac{3}{2}).\) Thus the slope is \(m=\frac{-3/2-0}{0-3}=\frac{1}{2}\). Thus the inclination is \(\tan^{-1}(\frac{1}{2})\approx 27^\circ.\)

    3. \(-4x+2y+1=0\)

      This equation is in general form so it is easiest to find the intercepts first. When \(y=0\), we get \(-4x+1=0\), and so the \(x\)-intercept is \((\frac{1}{4},0).\) When \(x=0\), we get \(2y+1=0\), and so the \(y\)-intercept is \((0,-\frac{1}{2}).\) Thus the slope is \(m=\frac{-1/2-0}{0-1/4}=2\). Thus the inclination is \(\tan^{-1}(2)\approx 63^\circ.\)

    4. \(10x-12y+3=0\)

      This equation is in general form so it is easiest to find the intercepts first. When \(y=0\), we get \(10x+3=0\), and so the \(x\)-intercept is \((-\frac{3}{10},0).\) When \(x=0\), we get \(-12y+3=0\), and so the \(y\)-intercept is \((0,\frac{1}{4}).\) Thus the slope is \(m=\frac{1/4-0}{0--3/10}=\frac{5}{6}\). Thus the inclination is \(\tan^{-1}(\frac{5}{6})\approx 40^\circ.\)

    5. \(9x+16y-225=0\)

      This equation is in general form so it is easiest to find the intercepts first. When \(y=0\), we get \(9x-225=0\), and so the \(x\)-intercept is \((25,0).\) When \(x=0\), we get \(16y-225=0\), and so the \(y\)-intercept is \((0,\frac{225}{16}).\) Thus the slope is \(m=\frac{225/16-0}{0-25}=-\frac{9}{16}\). Thus the inclination is \(\tan^{-1}(-\frac{9}{16})+180^\circ\approx 151^\circ.\)

  3. Give equations in intercept form for the following lines. Then use intercept form to find the intercepts of the line.

    1. the line through \((2,1)\) and \((3,2)\)

      Let's find the slope and use point-slope form. The slope is \(m=\frac{2-1}{3-2}=1,\) so an equation of this line is \(y-1=1(x-2).\) Now let's put the equation in intercept form: \[\begin{align} y-1 &= 1(x-2) \\ y-1 &= x-2 \\ -x+y &= -1 \\ \frac{-x}{-1}+\frac{y}{-1} &=1 \\ \frac{x}{1}+\frac{y}{-1} &=1. \end{align}\]

      From this form, we can see that \(a=1\) and \(b=-1.\) Thus the \(x\)-intercept is \((1,0)\) and the \(y\)-intercept is \((0,-1).\)

    2. the line through \((4,2)\) with slope \(3\)

      Let's use point-slope form. An equation of this line is \(y-2=3(x-4).\) Now let's put the equation in intercept form: \[\begin{align} y-2 &= 3(x-4) \\ y-2 &= 3x-12 \\ -3x+y &= -10 \\ \frac{-3x}{-10}+\frac{y}{-10} &=1 \\ \frac{x}{10/3}+\frac{y}{-10} &=1. \end{align}\]

      From this form, we can see that \(a=10/3\) and \(b=-10.\) Thus the \(x\)-intercept is \((10/3,0)\) and the \(y\)-intercept is \((0,-10).\)

    3. the line \(y=3x+5\)

      Let's put the equation in intercept form: \[\begin{align} y &= 3x+5 \\ -3x+y &= 5 \\ \frac{-3x}{5}+\frac{y}{5} &=1 \\ \frac{x}{-5/3}+\frac{y}{5} &=1. \end{align}\]

      From this form, we can see that \(a=-5/3\) and \(b=5.\) Thus the \(x\)-intercept is \((-5/3,0)\) and the \(y\)-intercept is \((0,5).\)

    4. the line \(4x+9y=36\)

      Let's put the equation in intercept form: \[\begin{align} 4x+9y &= 36 \\ \frac{4x}{36}+\frac{9y}{36} &=1 \\ \frac{x}{9}+\frac{y}{4} &=1. \end{align}\]

      From this form, we can see that \(a=9\) and \(b=4.\) Thus the \(x\)-intercept is \((9,0)\) and the \(y\)-intercept is \((0,4).\)

    5. the line \(y-2=3(x+1)\)

      Let's put the equation in intercept form: \[\begin{align} y-2 &= 3(x+1) \\ y-2 &= 3x+3 \\ -3x+y &= 5 \\ \frac{-3x}{5}+\frac{y}{5} &=1 \\ \frac{x}{-5/3}+\frac{y}{5} &=1. \end{align}\]

      From this form, we can see that \(a=-5/3\) and \(b=5.\) Thus the \(x\)-intercept is \((-5/3,0)\) and the \(y\)-intercept is \((0,5).\)

    2. \(y=\frac{3}{4}x+\frac{5}{6}\) to general form

    3. \(2x+3y+6=0\) to slope-intercept form

      Solve the current equation for \(y\): \[\begin{align} 2x+3y+6&=0 \\ 3y &= -2x-6 \\ y &= -\frac{2}{3}x-2. \end{align}\]

    4. \(\frac{2}{5}x+\frac{3}{5}y+20=0\) to slope-intercept form

    5. \(\frac{x}{2}+\frac{y}{3}=1\) to slope-intercept form

      Solve the current equation for \(y\): \[\begin{align} \frac{x}{2}+\frac{y}{3}&=1 \\ \frac{y}{3} &= -\frac{x}{2}+1 \\ y &= -\frac{3}{2}x+3. \end{align}\]

    6. \(\frac{x}{4}+\frac{y}{-2}=1\) to slope-intercept form

      Solve the current equation for \(y\): \[\begin{align} \frac{x}{4}+\frac{y}{-2}&=1 \\ \frac{y}{-2} &= -\frac{x}{4}+1 \\ y &= \frac{1}{2}x-2. \end{align}\]

      Set the current equation equal to zero and put the coefficients in front of the variables. One possible answer is \(\frac{1}{2}x+\frac{1}{3}y-1=0.\) Another is \(3x+2y-6=0.\)

    1. \(3(x-5)+2(y+7)=6\)

      We can simplify this to \(3x+2y=7,\) and from there we can see that the intercepts are \((0,7/2)\) and \((7/3,0).\)

    4. \(\frac{x}{-1}+\frac{y}{3}=1\)

      This is in intercept form, so we can see that the intercepts are \((-1,0)\) and \((0,3).\)

      This is in general form, so we can see that the intercepts are \((-3,0)\) and \((0,15/7).\)

    6. \(y-3=2(x-5)\)

      This is in point-slope form, so we can see that point \((5,3)\) is on the graph. Since the slope is \(2\), we can go over \(1\) and up \(2\) from \((5,3)\) to get \((6,5)\) also on the graph.
